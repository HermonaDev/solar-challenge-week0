# Solar Challenge Week 0

Cross-country solar farm analysis for Benin, Sierra Leone, and Togo.

## Environment Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/HermonaDev/solar-challenge-week0.git
   
   cd solar-challenge-week0"" 

## Development Workflow

This repository uses protected branches to ensure code quality:

1. Create a feature branch: `git checkout -b feature-name`
2. Make changes and commit: `git commit -m "description"`
3. Push and create PR: `git push -u origin feature-name`
4. **Requires:** 
   - ✅ One approval
   - ✅ Passing CI checks
   - ✅ Up-to-date with main
5. Merge via Pull Request
